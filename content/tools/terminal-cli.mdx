Modern terminal emulators and shell tools for enhanced productivity.

## Modern Terminal Emulators

### Warp
Modern, Rust-based terminal with AI command search and collaborative features.

**Key Features:**
- AI-powered command search
- Built-in command palette
- Blocks for organizing output
- Team collaboration features
- Native performance

[Visit Website →](https://www.warp.dev/)

---

### Ghostty
Fast, native, and feature-rich terminal emulator built with performance in mind.

**Key Features:**
- GPU-accelerated rendering
- Native platform integration
- Ligature support
- Customizable keybindings
- Low resource usage

[Visit Website →](https://ghostty.org/)

## Shell

### Zsh
Powerful Unix shell with advanced features, scripting capabilities, and customization.

**Key Features:**
- Advanced tab completion
- Spelling correction
- Themeable prompts
- Plugin architecture
- Compatible with Bash

[Visit Website →](https://www.zsh.org/)
