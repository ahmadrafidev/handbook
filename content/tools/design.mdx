UI design tools, CSS frameworks, and component libraries for building beautiful interfaces.

## Design Tools

### Figma
Collaborative interface design and prototyping tool with real-time collaboration capabilities.

**Key Features:**
- Real-time collaboration
- Component libraries
- Prototyping and animations
- Developer handoff tools

[Visit Website →](https://www.figma.com/)

---

### Paper
Digital sketching and wireframing tool for quick ideation and low-fidelity prototyping.

**Key Features:**
- Hand-drawn style wireframing
- Quick sketching capabilities
- Collaborative design
- Simple and intuitive interface

[Visit Website →](https://app.paper.design/)

---

### Excalidraw
Virtual whiteboard for sketching hand-drawn like diagrams and wireframes.

**Key Features:**
- Hand-drawn style
- Collaborative editing
- Open source
- Simple and fast

[Visit Website →](https://excalidraw.com/)

## CSS Frameworks

### Tailwind CSS
Utility-first CSS framework for rapid UI development with composable utility classes.

**Key Features:**
- Utility-first approach
- Responsive design utilities
- Customizable design system
- JIT compiler for optimization

[Visit Website →](https://tailwindcss.com/)

---

## Component Libraries

### Shadcn UI
Re-usable component collection built with Radix UI primitives and styled with Tailwind CSS.

**Key Features:**
- Copy-paste components
- Accessible by default
- Customizable styling
- TypeScript support

[Visit Website →](https://ui.shadcn.com/)

## Development Tools

### Storybook
Tool for building, documenting, and testing UI components in isolation from your app.

**Key Features:**
- Component isolation
- Interactive documentation
- Visual testing
- Addon ecosystem

[Visit Website →](https://storybook.js.org/)
